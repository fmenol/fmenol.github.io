## Light as a Control Input

Optogenetics gives us a remarkable ability: controlling protein activity with light. For cybergenetics, this means we can adjust gene expression levels with millisecond precision and no chemical interference.

The most common tools fall into two categories:

### Blue Light Systems

The **EL222** system from *Vibrio cholerae* has become a workhorse of cybergenetics:

- Activates in response to blue light (~450nm)
- Fast dynamics (seconds to minutes)
- Reversible upon light removal

```python
# Light intensity to expression mapping (simplified)
def expression_rate(light_intensity):
    k_max = 0.5  # maximum transcription rate
    K_d = 10     # half-saturation constant
    n = 2        # Hill coefficient

    return k_max * (light_intensity**n) / (K_d**n + light_intensity**n)
```

### Red/Far-Red Systems

The **PhyB-PIF** system offers orthogonal control:

- Red light (660nm) activates
- Far-red light (730nm) deactivates
- Useful for multiplexed control

## Practical Considerations

When setting up an optogenetic experiment, several factors matter:

1. **Light delivery**: LEDs vs. projectors vs. confocal
2. **Temporal resolution**: How fast can you update the signal?
3. **Spatial resolution**: Single-cell vs. population control
4. **Phototoxicity**: Blue light especially can damage cells
